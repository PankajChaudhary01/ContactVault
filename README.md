# ContactVault
ContactVault is a simple and efficient PHP-based Contact Management System that enables users to add, edit, delete, and search contacts. The project is built using PHP, MySQL, HTML/CSS, and Bootstrap for a clean and responsive interface.

## Features
- Add new contacts
- Edit contacts
- Delete contacts
- Search contacts by name or number
- Simple Bootstrap UI
- Easy database setup

## Technologies Used
- PHP
- MySQL
- HTML & CSS
- Bootstrap
- XAMPP (Localhost)
